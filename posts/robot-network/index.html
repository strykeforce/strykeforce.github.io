Robot Network

Oct 20, 2017

While using FRC radios in the default configuration is easy to set up, having to connect to an isolated robot access point is inconvienient during development when Internet access is required. We have installed OpenWRT on the Open Mesh OM5P-AC used on our robots.

Diagram

Open Mesh OM5P-AC Radio

The radio is configured using OpenWRT as a router with its WiFi interface connecting to our shop network with static IP address 192.168.3.251 and its ethernet ports on the 10.27.67.0/24 network.

Router

Our router is configured with a static route to the robot radio.
